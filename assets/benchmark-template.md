# 对标库表格模板（飞书多维表格字段设计）

本模板用于搭建个人对标库。飞书多维表格支持 AI 字段，可自动提取标题、内容、仿写。直接按以下字段在飞书新建多维表格即可。

## 一、账号对标库

| 字段名 | 类型 | 说明 / 示例 |
|--------|------|------------|
| 对标账号 | 单行文本 | 账号名称，如"数字生命卡兹克" |
| 账号情况概述 | 多行文本 | 规模、定位，如"AI工具定位、头部号" |
| 对标要素 | 多行文本 | 为啥对标，如"1、话题新；2、情绪化故事化写作好" |
| 其他思考 | 多行文本 | 变现模式、更新规律、团队情况等持续观察信息 |
| 状态 | 单选 | 活跃 / 已失效 / 观察中 |

**AI 字段（可选，飞书特性）：**
- AI 读简介：自动提取账号定位关键词
- AI 评价值：自动评估该账号可借鉴程度

## 二、爆款对标库

| 字段名 | 类型 | 说明 / 示例 |
|--------|------|------------|
| 对标文章 | 超链接 | 粘贴文章链接 |
| 所属账号 | 单行文本 | 账号名称 |
| 数据 | 多行文本 | 阅读量、互动量等 |
| 对标要素 | 多行文本 | 对标这篇文章的啥（标题/开头/结构/钩子） |
| 对标原因 | 多行文本 | 为啥对标这篇 |
| 原因拆解 | 多行文本 | 它好在哪、怎么做到的，给出自己的思考 |
| 如何学习 | 多行文本 | 知道好、知道怎么做到，如何应用到自己身上 |
| 状态 | 单选 | 已拆解 / 已失效 / 待拆解 |

**AI 字段（可选，飞书特性）：**
- AI 读标题：自动提取文章标题
- AI 读内容：自动提取文章核心内容
- AI 仿写：基于爆款结构自动生成仿写草稿

## 三、使用原则

- 对标库是给自己看的，合脚最重要，不要迷信别人的。
- 群聊暂存（面对面建群输4位密码）适合随手转发，定期整理进表格沉淀。
- 定期淘汰过时/落后样本，保持对标库鲜活。
