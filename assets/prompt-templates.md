# 提示词模板集

直接复制使用。所有提示词都遵循：明确角色、给足上下文、指定输出格式。

## 一、定位提示词

```
我想做一个微信公众号 AIP（AI + 个人 IP），请帮我做账号定位。

我的背景信息（尽量详细，细小优点也写）：
- 职业/身份：
- 已掌握的 AI 技能：
- 兴趣/热爱的事：
- 已有资源（人脉/工具/经验）：
- 每天可投入时间：

请给我 10 个细分定位方向，每个包含：目标人群、我能提供的价值、可变现方式。
注意：筛选思维——其中通常 3 个平庸、5 个合理、2 个让人眼前一亮，请标注哪 2 个最值得做。
```

**二次微调（定位太宽泛时）：**
```
请针对第 X 个定位，结合我的"AI技能"和"程序员"背景，给出一个更垂直、更细分的切入点。
```

## 二、写作提示词（带 IP 人设）

```
你是一位擅长写公众号 AIP 文章的写手。请基于以下人设和背景，写一篇关于[选题]的文章。

【人设】
- 我是谁：[姓名/昵称]
- 我的背景：[职业、AI技能、赛道]
- 我的风格：[真诚/文艺/利他/温润有力 等]
- 我的故事资产：[踩过的坑、成长轨迹、真实体验]

【文章要求】
- 开头用[展示结果/痛点直击/钩子]方式吸引
- 正文有用有料，穿插我的真实故事但不喧宾夺主
- 结尾用[互动/金句]方式
- 让读者看完很有收获
- 用 Markdown 格式输出，结构清晰，重点加粗

请先给出标题，再写正文。
```

## 三、爆款拆解提示词

**自由版：**
```
这是一篇爆款文章，请你帮我全方位进行拆解，分析其能够爆火的原因。
[粘贴文章标题+正文]
```

**9 维度版：**
```
这是一篇爆款文章，请你帮我全方位进行拆解，分析其能够爆火的原因。
分析后按照如下 9 个维度进行规整：
标题吸引力、开头代入感、内容流畅度、信息价值感（案例运用）、
结尾行动欲（互动、引流）、情绪曲线、认知颠覆点、行动触发器、其他爆火因素。
[粘贴文章标题+正文]
```

## 四、文章优化提示词（对比法）

```
文章一（我的对标文章）：
[粘贴对标文章全文]

文章二（我自己的文章）：
[粘贴自己的文章全文]

对比一下，这两篇公众号文章哪篇好？
有什么共同点和不同点？
请从标题、开头、结构、案例运用、互动设计、人设融入等维度帮我拆解，
并指出我这篇文章具体可以如何优化。
```

## 五、结尾升华提示词

```
请帮我为这篇文章升华一个不超过 300 字的结尾，要求有金句感、能引发读者共鸣和转发欲。
[粘贴文章正文]
```

## 使用提醒

- 提示词里加"让读者看完很有收获"能显著提升正文价值感。
- AI 写完后多交互一两轮挑选修改，避免啰嗦。
- 生成文章时，一并输出：推荐的 `assets/wechat-html-templates/` 模板名、1 张封面和 2 张文内图的生成提示词、每张图的 alt/图注/来源说明，以及一张 1200×630 宣传卡片文案。
- 朱雀 AI 检测可查 AI 痕迹，适当加入个人真实故事降低 AI 感。

## 六、图文模板与宣传卡提示词

```
请把下面这篇公众号文章打包成“可发布图文包”。

【文章正文】
[粘贴文章]

请输出：
1. 推荐模板：从 assets/wechat-html-templates/01-magazine-longform.html、02-tutorial-checklist.html、03-story-opinion.html 中选 1 个，并说明原因。
2. 模板数据块：标题、导语、作者、日期、段落、引用语。
3. 配图方案：1 张 2:1 封面 + 2 张正文图。每张必须给出用途、中文 alt、图注、素材来源/授权记录字段、可用于生图的中文提示词；发布时在公众号后台上传或替换图片，不依赖外部图片 URL。
4. 1200×630 宣传卡：为 04-promo-card.html 输出 tag、标题、摘要、链接文字和主视觉提示词。
5. 公众号粘贴说明：提醒用户打开所选模板，点击“复制公众号兼容排版”，再到后台粘贴并预览；该导出版本不包含 `script` 或 `style` 标签。

图片必须服务正文，不用无关氛围图；没有授权来源时明确标为“待补素材”。
```
