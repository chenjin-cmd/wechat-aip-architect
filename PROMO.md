刚开源了一个公众号 AIP 打造方法论仓库：**wechat-aip-architect**。

它不是某个 AI 工具的私有插件，而是一份**工具无关的知识包**——`SKILL.md` 给 WorkBuddy 自动触发，`UNIVERSAL.md` 给 Claude / ChatGPT / Cursor 直接喂上下文就能用。

覆盖定位 → 对标 → 选题 → 写作 → 排版全链路，5 个模块化 reference + 内容模板 + 4 套 HTML 图文模板。文章写完还能直接做封面、文内配图方案和 1200×630 宣传卡。想用 AI 把公众号跑成稳定内容流水线的，直接 clone 看。MIT 协议，随便改。
